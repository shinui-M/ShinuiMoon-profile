# 문신의 포트폴리오

영남대학교 화학공학부 에너지화공전공 문신의의 개인 포트폴리오 웹사이트입니다.

## 구성

- **소개** — 연구 방향 및 관심 분야
- **학력 및 수상** — 영남대학교 재학, 장학금 내역
- **연구 경험** — DGIST 하계 인턴십, 캡스톤 프로젝트
- **학술 활동** — 한국화학공학회 발표 이력
- **역량** — 전공, 실험 기술, AI 도구, 어학
- **과외 활동** — 동아리, 컨퍼런스
- **증명서** — 재학증명서, 성적증명서, TOEIC 성적표 다운로드

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4

## 로컬 실행

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000)에서 확인
